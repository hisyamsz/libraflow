## [1.0.0] - 2026-07-10
### Added
- Initial release of LibraFlow Library Management System.
- Backend using Express.js and Prisma.
- Frontend using Next.js and Tailwind CSS.
- Complete Docker Compose setup with MariaDB and phpMyAdmin.
- CI/CD pipeline via GitHub Actions.
- Comprehensive API Documentation (Swagger).
- Health check endpoints.
- Generic branding for open-source portfolio readiness.
